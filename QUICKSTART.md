# QUICKSTART

Быстрый старт теперь ведётся через [README.md](README.md).

Минимальный сценарий:

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
streamlit run main.py
```

Для полного описания команд, переменных окружения и возможностей интерфейса используйте [README.md](README.md).
